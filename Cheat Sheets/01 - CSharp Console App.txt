C# Console App Cheat Sheet
--------------------------
0. Make sure you have the following environment variables set on your machine:
	a. AzureOpenAIEndpoint
	b. AzureOpenAIKey
1. Open Program.cs and review code
	a. Kernel builder using Azure OpenAI chat completion service
	b. Semantic function requires prompt text and request settings
	c. Kernel function calls are asynchronous
2. Try it out
	a. Navigate to 01_CSharpConsole folder
	b. dotnet run
	c. Follow prompts and review results
