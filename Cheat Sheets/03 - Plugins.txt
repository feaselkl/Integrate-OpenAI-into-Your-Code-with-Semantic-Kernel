Plugins Cheat Sheet
-------------------
1. Open Program.cs and review code
	a. Note samplePlugins and customPlugins folders
	b. Review SamplePlugins and CustomPlugins folder contents: config.json and skprompt.txt for each
	c. SequentialPlanner: run one plugin after another
    d. Tasks to generate results (can't run all of these at once to prevent 429 errors!)
    e. Switch command to indicate the selected option
2. Try it out
	a. Navigate to 03_CSharpConsoleWithPlugins folder
	b. dotnet run
	c. Follow prompts and review results
